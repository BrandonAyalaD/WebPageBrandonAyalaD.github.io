Geo-Action Automator:

The Challenge: Designing a real-time notification system independent of paid third-party applications by bridging physical GPS location data with cloud-based workflows.
The Solution: I developed a custom infrastructure leveraging mobile Geofencing to trigger events via Webhooks to the GitHub API. Once the signal is received, a GitHub Actions workflow processes the metadata (location, timestamp, and movement type) and distributes automated notifications via SMTP to a defined group of recipients.
The Impact: A fully autonomous, cost-free, and scalable system that demonstrates the power of "headless" automation and seamless hardware-to-cloud integration.
